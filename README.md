## ğŸ§© Technical Details

### Built With
- **HTML5**: Semantic markup structure
- **CSS3**: Advanced styling with CSS variables, flexbox, and grid
- **JavaScript**: Vanilla JS for interactive functionality
- **Font Awesome**: Icons for UI elements
- **Google Fonts**: Persian typography support

### Key JavaScript Functions
- `loadStory()`: Manages narrative progression
- `applyEffects()`: Handles stat changes from choices
- `showPage()`: Page navigation system
- `saveGame()`/`loadGame()`: Save state management
- `changeTheme()`: Dynamic theme switching

### Data Structure
- Story nodes stored as JavaScript objects
- Choice-based branching narrative
- Player stats tracked in real-time
- Local storage for save games and settings
